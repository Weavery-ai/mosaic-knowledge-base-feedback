# Mosaic Knowledge Base Feedback

This repository is for collecting user feedback, bug reports, and feature requests for Mosaic Knowledge Base.

## How to Submit Feedback

1. Click on the **Issues** tab above
2. Click **New Issue**
3. Choose a template:
   - **Bug Report** - Report something that isn't working correctly
   - **Feature Request** - Suggest a new feature or improvement

## Adding Screenshots or Video

Both templates support uploading screenshots and screen recordings:
- Drag and drop images directly into the text area
- Paste screenshots from your clipboard
- Upload video files (MP4, MOV, etc.) for screen recordings

Visual evidence helps us understand and address issues faster.

## Response Time

We review feedback regularly and will respond to issues as soon as possible.
